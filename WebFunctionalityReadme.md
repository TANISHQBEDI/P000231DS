# Web App Notes

## Requirements

Node.js version v24.15.0 (LTS)
- https://nodejs.org/en/download

Dependencies
- requirements.txt


##  Running the web app

Step 1: run flask backend from root
- Run python backend/run.py

Step 2: run react frontend from /frontend
- cd to frontend
- npm run dev

Step 3: open up the front end
- should be localhost:5173
